Overview

Digital Junk Food Essentials is my first publicly released Unity project, created while completing the Unity Essentials learning pathway.

Rather than being a traditional game, this project serves as an interactive collection of scenes demonstrating core Unity concepts including:

-Player movement

-Cameras

-Physics

-Lighting

-Audio

-UI

-Animation

-Scene management

-Interactive environments
